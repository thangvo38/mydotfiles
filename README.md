# My dot files
My personal configuration files for vim, tmux and more

# Installation (fresh machine)
- Run `bash install.sh`

# Post-install
## For Vim
- Install Vundle
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

- Then run
```
vim +PluginInstall +all
```
